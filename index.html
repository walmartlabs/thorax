Quick Start

Thorax can be used standalone but is designed to work best with Lumbar. The easiest way to setup a new Thoax + Lumbar project is with the the Thorax command line library:

npm install -g thorax-cli
thorax create project-name

This will create a blank project. If you'd like to play with something that is already functional install the Thorax Todos application:

cd project-name
thorax install thorax-todos
jake init
jake start
